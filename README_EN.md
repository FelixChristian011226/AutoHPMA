**AutoHPMA** is a desktop tool built with C# and WPF, designed for *Harry Potter: Magic Awakened* to provide players with simple automation features.  

**Note**: This app only support Chinese version currently. Multi-language support is not yet available. I welcome any developers who would like to help with the development—please get in touch!

<br>

## Features

- **Auto Club Quiz**: Automatically participate in club quiz events, recognize questions via OCR, and match answers from the question bank.
- **Auto Forbidden Forest**: Automatically complete a specified number of Forbidden Forest runs, with auto-liking teammates on the summary page.
- **Auto Wizard Cooking**: Supports automated cooking workflows for multiple recipes. Users can customize or add recipe configurations.
- **Auto Sweet Adventure**: Automates the limited-time event "Sweet Adventure", including auto-matching, round progression, and settlement.

<br>

## Highlights

- Logging System: Overlay log display on the game window, with local log file support
- Mask Window: Customizable real-time overlay window showing detection results
- Hotkey Binding: User-defined hotkeys to quickly toggle features
- Notification: Native Windows notifications for real-time result feedback

## Installation

### Requirements

- Windows 10 or later
- .NET 8.0 or above

### Steps

1. Go to [Releases](https://github.com/FelixChristian011226/AutoHPMA/releases) to download the latest version.
2. Download and run `AutoHPMA-Setup.exe` to start the installer.
3. Follow the prompts to complete installation. You can customize the installation path and startup settings.
4. If prompted about missing .NET runtime on first launch, download it from [Microsoft .NET](https://dotnet.microsoft.com/download).

<br>

## Usage

1. Launch `AutoHPMA.exe`
2. Click the "Start" button on the main page to bind the game window
3. Navigate to the "Tasks" page to enable desired features

For detailed tutorials, FAQs, and more, visit the [official documentation](https://autohpma.top).

<br>

## Notes

- **Only compatible with MuMu emulator**, recommended resolution 1280×720. Other resolutions may cause screenshot issues.
- Set game graphics to the default "Standard" quality.
- Do not minimize the game window while the script is running, as it may cause window layering issues.
- This tool is developed for personal use and learning purposes only. Use at your own risk.
